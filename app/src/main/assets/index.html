Welcome to the WebServer sample!

If you have directory listing enabled, check out folder to check, if it is working!

Additionally, you can also check out the basic test file as test files are awesome.

You may wonder why the url of the linked files and this file differs.
That is pretty easy, the sample uses a router to link handlers to specific routes.
The /assets/ path is linked to a StaticAssetHandler, which serves files from the assets directory of the phone, while / and /index.html are statically linked to the assets/index.html file of the phone.
